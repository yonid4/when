## Backend:
### Activate Virtual Environment
```bash
.venv/Scripts/activate
# or
. .venv/bin/activate
```
### Run Server
```bash
python server.py
```

## Libraries Installation
```bash
cd .\.venv\
pip install -r requirements.txt
```
