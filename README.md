# WHEN

## Frontend:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Backend:
### Initialze Database
```bash
python create_db.py
```
### Run Server
```bash
flask run
python server.py
```

### Activate Virtual Environment
```bash
.venv/Scripts/activate
# or
. .venv/bin/activate
```

## Backend Libraries Installation
```bash
pip install -r requirements.txt
```

## Backend Libraries Installation
